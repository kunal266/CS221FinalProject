
Fist install the dependencies in a conda environment:

```shell script
conda create --name agario python=3.8
conda activate agario
pip install -r requirements.txt
```

Then run the RL Agent:

```shell script
python main.py
```

To run our Agar.io clone:

```shell script
python3 agario.py
```
